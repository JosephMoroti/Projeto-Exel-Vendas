# Projeto-Exel-Vendas


Este projeto consiste na criação de um dashboard analítico para monitoramento e acompanhamento das vendas de assinaturas do Xbox Game Pass ao longo de 2024. O principal objetivo foi transformar dados brutos em informações estratégicas, organizadas de forma visual e intuitiva, facilitando a análise do desempenho comercial e apoiando a tomada de decisões orientadas por dados. A solução permite acompanhar indicadores relacionados às assinaturas dos planos Xbox Game Pass, receitas geradas, renovações automáticas, adesão aos pacotes adicionais (EA Play e Minecraft Season Pass) e comportamento dos assinantes durante o período analisado.

Tecnologias e Recursos Utilizados
Microsoft Excel como fonte de dados; Tabelas Dinâmicas para consolidação e análise dos dados de vendas; Segmentações de Dados (Slicers) para filtragem interativa das informações; Análises de Negócio voltadas para faturamento, assinaturas, renovações automáticas e vendas de serviços complementares (EA Play Season Pass e Minecraft Season Pass).
